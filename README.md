# IAMOOT Admin Portal

Frontend application for IAMOOT organizers to manage both the written and oral competitions.

## Key responsibilities:

- Admin authentication
- Written competition management:
  - Submission status overview
  - Memorandum results and rankings
  - Penalty application and score adjustments
  - Detailed score breakdown per team and judge
- Oral competition management:
  - Preliminary match overview
  - Match details (teams, judges, speakers, scores)
  - Judge assignment (add/remove per match)
  - Winner selection
  - Preliminary team standings (wins/losses)
  - Semifinal match overview
  - Semifinal team results with role-based filtering (State/Victim)

Built with React and Vite, following the same architectural patterns used across IAMOOT frontends.
